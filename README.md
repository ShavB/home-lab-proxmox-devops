# Proxmox Lab for Devops

<details>
<summary>Overview</summary>

This section provides an overview of the Proxmox Lab project, including its purpose and key features.

A USB flash drive (16GB+ recommended) – for Proxmox installation.

A second USB or machine for booting the Proxmox ISO (or a way to boot from ISO, like Ventoy or Rufus-created USB).

Access to the Proxmox VE ISO: https://www.proxmox.com/en/downloads

A target system to host Proxmox (can be an old laptop, server, etc.)

</details>

<summary>Usage</summary>

Details on how to use the Proxmox Lab.

1. Download Proxmox VE ISO

2. Get it from the official Proxmox site.

3. Create a Bootable Installer USB

4. Use a tool like:

5. - Rufus (Windows)
   - Balena Etcher - balenaEtcher (Linux/macOS/Windows)

6. Write the Proxmox ISO to a USB drive.
